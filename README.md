# Final-Project-currency-conversion
In this project, my team and I created a program in which translates any currency the user inputs into the desired currency that the user wishes for. The code posted is the final, fully polished version of the code. For this code to function, 2 different API's are needed along with keys to use it. 

API key websites -------> https://openweathermap.org/api <----(OPEN WEATHER API) 
                 -------> https://rapidapi.com/principalapis/api/currency-conversion-and-exchange-rates <<<--------(Currency conversion and exchange rates API)
